Programming Languages Influence Network

This network graph visualization was created with data from the Freebase Programming Language collection and the JavaScript library sigma.js.

Help

Graph Navigation

You can zoom in and out the graph with the mouse wheel. You can move the graph by clicking and holding the left mouse button and moving the mouse.

Language Information

When you click on a language node in the graph a modal window with information about the language will be displayed.

Language Highlighting

Move the mouse over a language to highlight its connections. Languages in red influenced the highlighted language and blue ones were influenced by it.

Paradigm Highlighting

Choose and click on a paradigm from the Language Paradigm menu to only show languages within this paradigm.

Graph Layout

You can apply a Force Atlas 2 layout to the graph by clicking on Start Layout. Stop the layout by clicking on Stop Layout, before you further interact with the graph, e. g. highlight a language or paradigm.
